Comparison to New York

Visualisation by Raphael Canty

New York was hit fairly hard by COVID-19 upon the initial outbreak in early 2020. Unlike London, New York avoided subsequent lockdowns though a similar number of deaths per day were recorded at the start of 2021.

New York's COVID-19 response timeline is produced below, based on the same d3.js code as London's. You may scrub through events using the slider.

The data manipulation behind this visualisation is the work of Emily Wu and was done as part of the Compare City Crime project

Note that Google's data does not provide a breakdown for New York City, instead separating each state into counties. Therefore, only the inner city area of Manhattan could be extracted for the chart.

Data was sourced from:

  • New York COVID-19 Case and Death data: COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University
  • Apple: "Apple Maps COVID-19 Mobility Trends Report"
  • Citymapper: "Citymapper Mobility Index (CMI)"
  • Google: "Google COVID-19 Community Mobility Reports"
  • TomTom: "TomTom Traffic Index" obtained from ActiveConclusion's GirHub repository
  • Waze: "Waze COVID-19 Impact Dashboard"
