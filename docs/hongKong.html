Comparison to Hong Kong

Visualisation by Raphael Canty

Hong Kong, despite geographical vicinity to the original epicentre of the COVID-19 outbreak, did not endure the same level of cases as London.

Hong Kong's COVID-19 response timeline is produced below, based on the same d3.js code as London's. You may scrub through events using the slider.

The data manipulation behind this visualisation is the work of Emily Wu and was done as part of the Compare City Crime project

Use the slider to look through events

All data sources used for the London mobility change chart are available for Hong, with the exception of Transport for London's data which only applies to public transport in London, and Waze which did not publish data for Hong Kong.

Below is a Hong Kong's mobility change chart, where you may enable or disable each individual source of data. Note that a full lockdown was implemented in Sydney on 28 July, just before the end of this data series.















Data was sourced from:

  • Citymapper: "Citymapper Mobility Index (CMI)"
  • Google: "Google COVID-19 Community Mobility Reports"
