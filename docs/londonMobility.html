Visualising London's Covid-19 Mobility Change

Visualisation by Raphael Canty

Several public and private-sector datasets have been made available which measure mobility change of London during the COVID-19 Pandemic. This interactive visualisation was created using d3.js and is intended to display a collection of these datasets together for them to be compared and contrasted over time.

To display on the chart, each dataset's format was standardised to a percentage score of 100: some recorded this as percentage change from baseline (-25% = 75%) or as a fraction (0.75 = 75%). The TfL Cycle Hire category was created for this visualisation using TfL usage statistics. Data was taken from when it's first available until 31 July 2021.

A measure of restriction stringency is also included, with lockdown periods highlighted so that mobility can be visualised in the context of city-wide public health interventions. It is interesting to analyse how they shaped the curves.

Click each category to turn it on or off. This allows comparison between a smaller (and more readable) number of categories. Hovering over the chart displays the date and the readings for each category.
